Autonomous Forklifts (AFL) Nov 2024 · Oct 2025
Mobyus Future Tech R&D Lab
Demo video: Autonomous Forklifts (AFL)
  • Developed Frenet-frame optimal-trajectory planning with B-spline smoothing for obstacle avoidance and precise docking; implemented a three-wheeled controller on AFL (SD-type motors).
  • Built a local SLAM mapping + localization pipeline integrating LiDAR/IMU with ESKF; tuned for stable pose and reliable warehouse navigation; added PLC-triggered behaviors for production workflows.
Demo video: Obstacle avoidance (RealSense D435f + 2D SICK LiDAR)
  • Designed and developed obstacle detection using sensor fusion of a RealSense D435f depth vision sensor and a 2D SICK LiDAR to robustly detect obstacles for safe warehouse navigation.
Demo video: Pallet stack perception (SICK Visionary-T Mini) + truck space detection (ZED2i)
  • Developed pallet-stacking perception using a SICK Visionary-T Mini: estimated pallet size and fork-pocket (hole) centers with a Transformer-based model, leveraging CAD-based geometry to localize hole centers for precise insertion/alignment.
  • Built truck/trailer free-space perception using ZED2i stereo and object detection models (YOLOv12, RF-DETR) to estimate available loading space and placement feasibility.
